# 🌌 Neon Math - Interactive Glassmorphism Calculator

A sleek, modern, and highly interactive calculator built with **HTML5**, **CSS3 (Custom Properties & Animations)**, and **Vanilla JavaScript**. This project features a stunning glassmorphism UI and a dynamic floating math-symbol background animation.

## ✨ Features

-   **Glassmorphism UI:** A frosted-glass effect using `backdrop-filter`.
-   **Neon Aesthetic:** Cyberpunk-inspired colors with neon glows and shadows.
-   **Floating Animations:** Dynamic background with floating mathematical symbols generated via JavaScript.
-   **Responsive Design:** Works perfectly on different screen sizes.
-   **Standard Operations:** Supports Addition, Subtraction, Multiplication, Division, and Percentage.
-   **Smart Calculation:** Uses a secure `Function` constructor for evaluating expressions.

## 🛠️ Built With

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, CSS Variables, and Keyframe Animations.
* **Bootstrap 5:** For base styling components.
* **JavaScript (ES6):** Logic for calculations and background particle effects.

## 📂 Project Structure

- `index.html` - The structure and Bootstrap integration.
- `style.css` - Custom neon styling and animations.
- `script.js` - Calculation logic and particle system.
